Automatically assigns random publish dates to your draft posts within a specified date range and interval.

== Description ==

Random Date Changer is a WordPress plugin that helps you automate the publication of your draft posts by assigning them random publish dates within a date range you specify. Perfect for bloggers and content managers looking to manage their content schedule with ease.

**Features:**

- Set a date range for random publish dates.
- Specify the gap (in days) between consecutive publish dates.
- Automatically publish all draft posts with the assigned dates.
- Simple and intuitive admin interface.

Created by [Tuhin Bairagi](https://tuhinbairagi.com/).

== Installation ==

1. Log in to your WordPress admin panel.
2. Go to `Plugins > Add New`.
3. Search for "Random Date Changer" in the plugin directory.
4. Click `Install Now` next to the plugin.
5. Once installed, click `Activate`.
6. Navigate to `Random Changer` in the admin menu to start using the plugin.

== Frequently Asked Questions ==

= What happens if my date range is too short for the number of draft posts? =

The plugin will stop assigning dates once the range is exhausted.

= Can I undo the changes after assigning dates? =

No. Once the posts are published, changes cannot be reverted automatically. You will need to manually edit the posts.

= Does this plugin work with custom post types? =

Currently, it works only with standard posts. Support for custom post types may be added in the future.
